# Projeto Triângulo com JUnit

## Resumo dos Passos

1. **Criar o projeto Maven**  
   Utilizando o seguinte comando:
   ```bash
   mvn archetype:generate
   ```

2. **Adicionar as dependências do JUNIT no projeto**


3. **Rodar os testes**  
   Utilizando o seguinte comando:
   ```bash
   mvn test
   ```
